 This program is designed to work with custom hardware to act as data logger and central control ECU for a 1983 Honda CB1100F. It is written to run on a raspberry pi zero W and communicate with arduinos and other sensors to control and record various aspects of the motorcycle\textquotesingle{}s operation.\hypertarget{index_autotoc_md1}{}\doxysection{Componenents}\label{index_autotoc_md1}
